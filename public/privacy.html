Kebijakan Privasi

Efektif sejak: Januari 2026

Privasi Anda adalah prioritas utama Aurora Store. Dokumen ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data pribadi Anda saat menggunakan layanan kami.

1. Data yang Kami Kumpulkan

Kami hanya mengumpulkan data yang diperlukan untuk pengiriman dan transaksi. Data tersebut meliputi:

  • Username Pi: Untuk identifikasi akun Anda.
  • Alamat Pengiriman: Data fisik yang Anda masukkan untuk keperluan pengiriman barang.
  • Riwayat Transaksi: Digunakan untuk layanan purna jual dan klaim garansi.

2. Pi SDK dan Autentikasi

Aurora Store menggunakan Pi SDK resmi untuk otentikasi pembayaran demi keamanan maksimal:

  • Kami TIDAK pernah menyimpan Private Key (Passphrase) wallet Pi Anda di server kami.
  • Transaksi pembayaran ditandatangani langsung di perangkat Anda melalui Pi Browser/App.
  • Kami hanya menerima konfirmasi transaksi yang valid dari jaringan Pi.

3. Keamanan Data

Kami menerapkan enkripsi SSL standar industri (HTTPS) untuk melindungi data yang dikirim antara perangkat Anda dan server kami. Namun, metode transmisi melalui internet memiliki risiko inheren, dan kami tidak dapat menjamin keamanan mutlak dari peretasan pihak ketiga.

4. Cookies

Website ini menggunakan cookies teknis untuk menjaga sesi login Anda dan meningkatkan pengalaman pengguna (UX). Anda dapat mengatur browser Anda untuk menolak cookies, namun hal ini dapat mempengaruhi fungsionalitas website seperti keranjang belanja.
